# python-pptx
Python library for manipulating PowerPoint files. View the library's documentation [here](https://python-pptx.readthedocs.io/en/latest/).

### python_pptx_demo.py
Basic demo of python-pptx functionalities including:
- Creating presentation files
- Adding slides, textboxes, and shapes
- Creating tables, charts, and pictures

### superstore_example.py
Business use case to create quarterly sales presentation deck from [superstore dataset](https://www.kaggle.com/datasets/rohitsahoo/sales-forecasting).
